# Advent Of Code

[Advent of Code](https://adventofcode.com) is an annual set of Christmas-themed computer programming challenges that
follow an Advent calendar. The programming puzzles cover a variety of skill sets and skill levels and can be solved
using any programming language.

**_SPOILER ALERT_**: This repository contains my solutions to the Advent-Of-Code challenges.

**_Purpose_**: For me, these challenges present a really cool way to explore new programming language or programming
ideas. An exploration of new concepts and ideas by jumping in the deep end. AoC presents a very different way than the
likes of LeetCode, where the challenges don't have time based answers, and exploration into better solutions is
completely dependant on the participant.
