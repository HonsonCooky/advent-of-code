# Advent Of Code

[Advent of Code](https://adventofcode.com) is an Advent calendar of small programming puzzles for a variety of skill
sets and skill levels that can be solved in any programming language you like.

**_SPOILER ALERT_**: This repository contains my solutions to the Advent-Of-Code challenges.
Whilst I haven't kept a record what the challenges have asked for, you can find them by altering the URL for a
particular year, and going to the `calendar` page.
